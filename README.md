# Qt Calculator (MVC)
## Проект для портфолио | C++17 | QT

Простое приложение-калькулятор, разработанное на Qt и C++ с использованием архитектурного паттерна Model-View-Controller (MVC).

![](https://github.com/Rinalii/calculator-cpp-qt/tree/main/data/calculator_view.gif)

## ⚙️ Сборка и запуск
### Требования
- C++17
- QT 5.15

## 🧱 Возможности

+ Базовые арифметические операции: сложение, вычитание, умножение, деление.
+ Ввод десятичных чисел (с точкой).
+ Кнопка сброса AC (All Clear).

+ Два дисплея:
  + верхний — текущее выражение;
  + нижний — активное (вводимое) число.

+ Унарные операторы (+/- и %) 
+ Обработка ошибок — в разработке.

+ Поддержка клавиатуры:
  + цифры и точка;
  + операторы +, -, *, /;
  + = (Enter/Return);
  + AC (Escape);
  + % (процент).

+ Адаптивный интерфейс: кнопки меняют цвет при наведении и нажатии.

## 🧱 Архитектура
Проект строго разделён на три слоя согласно паттерну MVC:

+ Модель (CalculatorModel) — содержит всю логику вычислений, обрабатывает ввод пользователя, хранит состояние (предыдущее число, текущее число, оператор, флаг десятичной точки). Модель не зависит от Qt и использует только стандартную библиотеку C++.
+ Представление (Calculator) — графический интерфейс, построенный на виджетах Qt (QLabel, QPushButton, QGridLayout). Отображает выражение и текущее число, а также испускает сигналы при нажатии кнопок.
+ Контроллер (CalculatorController) — связывает модель и представление. Получает сигналы от кнопок, преобразует их в команды для модели и обновляет представление в соответствии с результатами.

Такое разделение упрощает поддержку, тестирование и расширение функциональности.
